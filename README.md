# Sales & Profit Analysis Dashboard
## Project Overview
This project is an interactive Power BI dashboard created to analyze sales and profit performance using the Superstore dataset.
## Key Features
 - KPI Cards :Total Sales,Profit,Quantity,Orders
 - Sales Trend Over Time(Line Chart)
 - Sales by Category
 - Profit by Sub-Category
 - Top Products Analysis
 - Sales vs Profit(Scatter plot)
 - State-wise Sales Map
 - Interactive Slicers for filtering by Date,Category,Segment, and State
## Insights
  -Technology category has the highest sales.
  -Some sub-categories generate lower profit.
  -Sales show an increasing trend over time.
## Tools Used
  -Power BI
  -CSV Dataset
